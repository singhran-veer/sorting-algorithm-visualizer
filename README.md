# 🧠 Sorting Visualizer

A web-based visualization tool that demonstrates how popular sorting algorithms work using animated bar charts.

## 📌 Features

- 🎯 Visualize step-by-step sorting
- ⚙️ Choose from:
  - Bubble Sort
  - Selection Sort
  - Quick Sort
  - Merge sort
- 🔢 Set custom array size
- 🔁 Generate random arrays
- 🧩 Clean, responsive UI
